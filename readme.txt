== Description ==

UserAgent Theme Switcher is a simple plugin which lets you change the theme to be displayed according to the detected browser.

Checks from a predefined list useragent so that the user can at will change the theme shown for that browser.
A user can configure a special theme for example for mobile phones that is lighter and this adapted to these devices

Ideal for displaying themes ready to opera mobile, internet explorer mobile, iphone, android, etc.

== Installation ==

Upload the UserAgent Theme Switcher plugin to your blog, Activate it
